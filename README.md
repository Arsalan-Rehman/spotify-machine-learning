## 🌟 Project Overview
This repository contains a comprehensive machine learning tutorial focusing on the **Random Forest Regressor**. The project goes beyond basic prediction to explore the **Bias-Variance Tradeoff** and **Explainable AI (XAI)** using a dataset of 114,000 Spotify tracks.



## 🛠️ Key Technical Features
- **Ensemble Learning:** Implementation of Bootstrap Aggregation (Bagging) to reduce model variance.
- **Complexity Analysis:** Deep dive into `max_depth` using Validation Curves to identify the transition from underfitting to overfitting.
- **Interpretability:** Use of Global Feature Importance and Partial Dependence Plots (PDP) to visualize "Black Box" decision-making.
- **Big Data Visualization:** Utilization of Hexbin Density plots to manage overplotting in large-scale datasets.



## 📂 Repository Structure
- `spotify.ipynb`: The complete Jupyter Notebook containing the narrated tutorial and Python code.
- `Final_Report.pdf`: A professional academic report summarizing the methodology and findings.
- `dataset.csv`: The processed Spotify Tracks dataset.
- `correlation_heatmap.png`: Diagnostic plot of feature relationships.
